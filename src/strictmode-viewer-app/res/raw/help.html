This application is designed to view violations in an easy and convenient way. It aggregates information on violations with same stacktraces and provides some basic statistics.

If you want to analyze StrictMode violations of your application you need to do some extra steps. Detailed step-by-step instructions can be found in this tutorial. You may want to consult wiki if you have more questions.

In case you want to quickly test what this application is capable of you can install StrictMode Violator application. You will be able to make as many violations as possible and view them using this application - StrictMove Viewer.
